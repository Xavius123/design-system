## Usage

### Web (CSS)

```css
/* Import tokens */
@import '@toyota/design-tokens/css/light';
@import '@toyota/design-tokens/css/dark';

/* Use in your styles */
.my-component {
  color: var(--color-light-primary);
  background: var(--color-light-background);
}

[data-theme="dark"] .my-component {
  color: var(--color-dark-primary);
  background: var(--color-dark-background);
}
```

### React Native (JavaScript)

```javascript
import { lightColors, darkColors } from '@toyota/design-tokens/js/light';
import { useColorScheme } from 'react-native';

const MyComponent = () => {
  const colorScheme = useColorScheme();
  const colors = colorScheme === 'dark' ? darkColors : lightColors;
  
  return (
    <View style={{ backgroundColor: colors.background }}>
      <Text style={{ color: colors.primary }}>Hello</Text>
    </View>
  );
};
```

## Token Files

Tokens are defined in JSON format in `packages/tokens/`:

- `colors-light.json` - Light theme colors
- `colors-dark.json` - Dark theme colors  
- `theme-light.json` - Light theme aliases
- `theme-dark.json` - Dark theme aliases

## Figma Integration

Colors are automatically synced from Figma using the Azure Pipeline. See `scripts/fetch-figma-tokens.js` for details.
