# Spacing Tokens

Consistent spacing scale for layout and component spacing.

## Spacing Scale

<div style={{ display: 'flex', flexDirection: 'column', gap: '1rem' }}>
  {[
    { name: 'xs', value: '0.25rem', px: '4px' },
    { name: 'sm', value: '0.5rem', px: '8px' },
    { name: 'md', value: '1rem', px: '16px' },
    { name: 'lg', value: '1.5rem', px: '24px' },
    { name: 'xl', value: '2rem', px: '32px' },
    { name: '2xl', value: '3rem', px: '48px' },
    { name: '3xl', value: '4rem', px: '64px' },
  ].map(({ name, value, px }) => (
    <div key={name} style={{ display: 'flex', alignItems: 'center', gap: '1rem' }}>
      <div style={{ width: '80px', fontWeight: 600 }}>{name}</div>
      <div style={{ width: '100px', fontSize: '0.875rem', color: '#6B7280' }}>{value} ({px})</div>
      <div style={{ 
        height: '32px', 
        width: value, 
        background: 'var(--color-light-primary)', 
        borderRadius: '4px' 
      }} />
    </div>
  ))}
</div>

## Usage

### Component Spacing

Use spacing tokens for consistent padding and margins in components:

```css
/* Button padding */
.button-sm {
  padding: var(--token-light-spacing-sm) var(--token-light-spacing-md);
}

.button-md {
  padding: var(--token-light-spacing-md) var(--token-light-spacing-lg);
}

.button-lg {
  padding: var(--token-light-spacing-lg) var(--token-light-spacing-xl);
}
```

### Layout Spacing

```css
/* Container padding */
.container {
  padding: var(--token-light-spacing-xl);
}

/* Stack spacing */
.stack > * + * {
  margin-top: var(--token-light-spacing-md);
}

/* Grid gap */
.grid {
  gap: var(--token-light-spacing-lg);
}
```

## Web (CSS)

```css
/* Import tokens */
@import '@toyota/design-tokens/css/light';

.my-component {
  padding: var(--token-light-spacing-md);
  margin-bottom: var(--token-light-spacing-lg);
  gap: var(--token-light-spacing-sm);
}
```

## React Native

```javascript
import tokens from '@toyota/design-tokens/js/light';

const styles = StyleSheet.create({
  container: {
    padding: parseInt(tokens.spacing.md),
    marginBottom: parseInt(tokens.spacing.lg),
    gap: parseInt(tokens.spacing.sm),
  },
});
```

## Spacing Guidelines

### Component Internal Spacing

- **xs (4px)**: Icon spacing, very tight elements
- **sm (8px)**: Compact components, dense layouts
- **md (16px)**: Default component padding
- **lg (24px)**: Comfortable component padding

### Layout Spacing

- **lg (24px)**: Component-to-component spacing
- **xl (32px)**: Section padding, card padding
- **2xl (48px)**: Page section spacing
- **3xl (64px)**: Major section dividers

## Token Reference

| Token | Value | Pixels | Common Use |
|-------|-------|--------|------------|
| `--token-light-spacing-xs` | 0.25rem | 4px | Icon gaps, minimal spacing |
| `--token-light-spacing-sm` | 0.5rem | 8px | Compact padding, tight gaps |
| `--token-light-spacing-md` | 1rem | 16px | Default padding, standard gaps |
| `--token-light-spacing-lg` | 1.5rem | 24px | Component spacing, comfortable padding |
| `--token-light-spacing-xl` | 2rem | 32px | Section padding, card padding |
| `--token-light-spacing-2xl` | 3rem | 48px | Page sections, large spacing |
| `--token-light-spacing-3xl` | 4rem | 64px | Major sections, hero spacing |
